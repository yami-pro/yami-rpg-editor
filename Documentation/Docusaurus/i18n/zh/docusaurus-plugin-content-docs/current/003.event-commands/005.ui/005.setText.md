# 设置文本

![](img/setText-1.png)

- 元素：文本元素访问器
- 属性列表：可以同时设置多个文本元素的属性

![](img/setText-2.png)

### 文本属性

- 属性
  - 内容：通过右键菜单插入富文本标签和变量
  - 字体大小
  - 行间距
  - 字间距
  - 颜色
  - 字体
  - 文字效果
    - 无
    - 阴影
    - 描边
    - 轮廓
  - 混合模式
    - 正常
    - 加法
    - 减法
