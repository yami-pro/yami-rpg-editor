# 文件浏览器

| 快捷键                        | 描述               |
| ----------------------------- | ------------------ |
| Up                            | 向上选择文件       |
| Down                          | 向下选择文件       |
| Left                          | 向左选择文件       |
| Right                         | 向右选择文件       |
| Ctrl + Left Mouse Button      | 多选/取消选择      |
| Shift + Left Mouse Button     | 选择连续的多个文件 |
| Ctrl + Wheel                  | 切换文件视图模式   |
| Ctrl + Drag                   | 复制选中的文件     |
| Backspace / Back Mouse Button | 返回上一层目录     |
