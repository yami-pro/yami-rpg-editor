# 全局

| 快捷键           | 描述                     |
| ---------------- | ------------------------ |
| Alt + 1          | 切换到第1个标签页        |
| Alt + 2          | 切换到第2个标签页        |
| Alt + N          | 切换到第N个标签页        |
| F1               | 打开"项目设置"窗口       |
| F2               | 重命名                   |
| F3               | 打开"全局变量"窗口       |
| F4               | 运行游戏                 |
| F6               | 打开"对象属性"窗口       |
| F7               | 打开"枚举"窗口           |
| F8               | 打开"过渡"窗口           |
| F9               | 打开"插件"窗口           |
| F10              | 打开"自定义事件指令"窗口 |
| F11              | 切换全屏模式             |
| F                | 水平翻转选中的图块       |
| Ctrl + N         | 新建项目                 |
| Ctrl + O         | 打开项目                 |
| Ctrl + S         | 保存项目                 |
| Ctrl + Shift + T | 重新打开关闭的标签页     |
