# 事件指令列表

| 快捷键          | 描述                 |
| --------------- | -------------------- |
| Up              | 选择上一个指令       |
| Down            | 选择下一个指令       |
| Enter           | 编辑                 |
| Space / Insert  | 插入                 |
| /               | 启用/禁用选中的指令  |
| \               | 插入脚本             |
| A - Z           | 搜索事件指令         |
| PageUp          | 向上翻页并选择指令   |
| PageDown        | 向下翻页并选择指令   |
| Ctrl + PageUp   | 向上翻页             |
| Ctrl + PageDown | 向下翻页             |
| Home            | 滚动到顶部并选择指令 |
| End             | 滚动到底部并选择指令 |
| Ctrl + Home     | 滚动到顶部           |
| Ctrl + End      | 滚动到底部           |
| Ctrl + A        | 全选                 |
| Ctrl + Z        | 撤销                 |
| Ctrl + Y        | 重做                 |
